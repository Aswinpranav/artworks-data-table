# Artworks Data Table

This project displays artworks from the Art Institute of Chicago API using React and TypeScript.

## Features
- Server-side pagination
- Multi-row selection using checkboxes
- Selection persists across pages
- Data fetched page-by-page from the API

## Tech Stack
- React
- TypeScript
- Vite
- PrimeReact

## API Used
https://api.artic.edu/api/v1/artworks

## Notes
- Only the current page data is stored in memory
- Selected rows are tracked using artwork IDs
